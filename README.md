AI Fashion Model은 Stable Diffusion을 활용하여 가상 패션 모델과 배경을 생성하는 프로젝트입니다.

## 🚀 기능
- AI 기반 가상 패션 모델과 배경 생성
- 프롬프팅으로 원하는 인종, 성별, 헤어스타일, 배경 선택 가능
